# Student Analytics API

This project is a Flask-based REST API designed to evaluate student performance, update grades securely, and generate student analytics.

## Features
- Evaluate student performance
- Calculate grade averages
- Update grades using token authentication
- Generate analytics summaries
- REST API endpoints with JSON responses

## Technologies Used
- Python
- Flask
- GitHub

## API Endpoints
- POST /api/students/evaluate
- PUT /api/students/update-grades
- GET /api/students/analytics

## Purpose
This project was created to demonstrate backend API development, request handling, authentication, and analytics processing using Flask.
